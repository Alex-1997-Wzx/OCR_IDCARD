# 基于OCR的身份证要素提取设计与实现

## 1.项目背景：[基于OCR识别技术的身份证要素提取](https://www.datafountain.cn/competitions/346)

## 2.环境版本说明：
	- Anaconda3: 4.7.12
	- PyCharm: 2019.3.3
	- OpenCV: 3.1.0
	- Numpy: 1.12.1
	- 详细组件版本信息请查看conda list文件

## 3.模块说明：
- detection_card_module: 定位身份证顶点坐标
- detection_text_module: 定位文本行并切分图片
- recognitio_words_module: 识别文本行信息
